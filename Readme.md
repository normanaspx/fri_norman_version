# Norman Vicente Unsplash App


## Arquitectura
- Kotlin
- Patron MVVM
- Injeccion de Dependencias Hilt 
- Room Database 
- Recyclerview

## Caracteristicas
- Buscador de imagenes consumiendo la API de unsplash
- Opcion de guardar la imagen sin conexion
